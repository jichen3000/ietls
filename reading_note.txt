 

阅读
  步骤：
    1、read the instruction
    2. underline the keywords
    3、有条件的对题目空格进行分析
    4、根据keyword，寻找答案。 注意同义转换
    
  keywords:
    1、专有名词 人名、地名
    2、数字  日期、货币
    3、特殊形容词
    4、特殊动词
    
  Skimming
    1、句子
    2、段落 抓住主题句，首句和尾句
    3、文章  结构、风格
  Scanning
    找答案， reading intensively
    
  技巧 could,might 表示委婉的一般对。 表示绝对的一般是错的。
  
  十类题型：
    1. sentence completion
    2. Summary  ++
    3. TURE|FALSE|NOT GIVEN ++
    4. List of Heading ++
    5. Multiple Choice ++ 单选
    6. Picking From a list 多选
    7. Classification
    8. Mathing ++
    9. Note|Table|Flow Chart completion
    10. Short Answer questions
    
    Matching的技巧：
      1、在文中找位置
      2、在答案中划keywords
      3、答案就在keywords的两句之内
      
    对于文章意思的理解，要大于任何技巧

判断题：
  定位词
  判断词
  方向词
  信号词
  
判断词：
  1. 谓语及之后
  2. be动词及之后
  3. 绝对化概念 most及最高级，best, only, first, all, fully, every, immediately
  4. 数字
  5. 比较关系， 定位词是双方，如果没有定位词，选NG
  6. 因果关系
  
判断题流程
  定位词 （无）-> NOT GIVEN
    |（有）
  判断词 （无）-> NOT GIVEN
    |（有）
  是否一致 （否）-> FALSE OR NO
    |（是）
  TRUE OR YES
  
文章：喜新厌旧，引出旧观点，反驳它。

选择题：
  错误选项
    1. 存在没有的信息
    2. 无
    3. 反
    4. 比较关系 通常不对（单选题）
    5. 程度副词(ly) 通常不对（单选题）
    6. 移花接木
    7. 遗漏关键信息
  正确选项
    1. A+B的情况， 因为内容比较完整
    2. May 可能的语气
    
填空题：
  summary 有选项的和没有选项的，有选项的相对难，答案和关键词都会被替换
  图表 容易
  完成句子
  简答
  
  形式：
    1. prep + __ 介词加空格
    查找  prep'+ n
    2. v + __
    查找 v' + n
    3. A and B
    4. A of B
      B 修饰 A
      A of B of C, C修饰B,B修饰A, = A of B which C = A which B that C
    5. 主动被动替换
    
语法
  A of B of C, C修饰B,B修饰A, = A of B which C = A which B that C
      
  某些动词可以接双宾语
    give sth to sb  (这时的sth只是简单的结构)
    give sb sth (这时的sth一般是复杂结构)
    give the book which is important to me. (to me是which从句的)
